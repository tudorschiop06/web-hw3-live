Portofoliu Personal Web

Acesta este un proiect de prezentare a parcursului meu academic si a proiectelor realizate.

  • Tehnologii utilizate: Dezvoltat folosind HTML5 si CSS3
  • Accesibilitate: Include functii precum "Skip to content" si utilizarea corecta a etichetelor
  • Design: Interfata este optimizata pentru a fi lizibila.

Manager de Parole

Aceasta este o aplicatie conceputa pentru a rezolva problema memorarii zecilor de parole diferite.

  • Stocare Centralizata: Toate conturile si parolele sunt salvate intr-un singur loc, fiind protejate de Master Password.
  • Securitate prin Criptare: Datele nu sunt salvate ca text simplu, ci sunt transformate intr-un cod imposibil de descifrat.
  • Protectie Offline: Baza de date ramane pe calculatorul utilizatorului, reducand riscul de a fi furata.

Aplicatie de Encriptare

Proiect ce implica confidentialitatea datelor, transformand mesajele in text cifrat.

  • Encriptare: Transforma textul brut intr-un cod ilizibil folosind o parola cheie;
  • Decriptare: Recupereaza informatia originala doar prin introducerea cheii corecte;
  • Securitate: Parolele nu sunt stocate, asigurand confidentialitatea totala a datelor.
